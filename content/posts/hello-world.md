## 一些基本功能

- **代码高亮**：行号、复制按钮、暗色模式都自带。
- **数学公式**：需要时在 `hugo.toml` 里加 `[params.math] enable = true`。
- **搜索**：PaperMod 自带 fuse.js 全文搜索，部署后直接能用。

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("Hugo"))
```
